To install:

Copy these snippets to C:\Users\<Username>\Documents\Visual Studio 2012\Code Snippets\Visual C#\My Code Snippets

To use:

Type one of these words and hit tab twice:

act -> Create ActionMethod
admincontroller -> Create an AdminAPIController
adminservice -> Create an AdminService class that inherits from AdminServiceBase
dto -> create a DTO object
dtoprop -> create a property on a DTO Object
map -> create an automapper mapping
